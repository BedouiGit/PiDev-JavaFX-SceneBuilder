NFT Marketplace Application
Overview
This JavaFX application, along with Scene Builder, serves as a platform for buying and selling Non-Fungible Tokens (NFTs). NFTs are unique digital assets that represent ownership of a particular item, artwork, or collectible. This marketplace provides users with the ability to browse, purchase, and list NFTs for sale.

Features
User Authentication: Secure user authentication system to ensure only authorized users can access the marketplace.
Browse NFTs: Users can browse through a collection of available NFTs, each with its unique details and attributes.
Purchase NFTs: Users can buy NFTs listed for sale by other users using a secure payment system.
Sell NFTs: Users can list their own NFTs for sale, setting their desired price and other relevant details.
User Profile: Each user has a profile where they can view their transaction history, manage their listings, and update their account information.
Search Functionality: Users can search for specific NFTs based on various criteria such as name, category, or artist.
Technologies Used
JavaFX: Java library for building desktop applications with rich graphical user interfaces.
Scene Builder: Drag-and-drop interface builder for designing JavaFX applications.
MySQL: Database management system for storing user data, NFT details, and transaction history.
Java Cryptography Architecture (JCA): Ensures secure communication and data encryption within the application.
Maven: Project management and build automation tool for Java projects.
